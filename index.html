Project Name: Recipee

Project Type: Food Recipe Sharing Website

Description:

Recipee is an innovative web application built using React.js and Django with the Django REST framework, tailored for sharing, discovering, and creating an extensive collection of food recipes. It provides a rich array of features to enrich the user experience.

Features:

  • User registration and secure authentication using JWT
  • Create, edit, and delete food recipe posts
  • User can comment and like the post
  • Bookmark favorite posts for quick access
  • User profile customization, including updating profile photos and mobile numbers
  • Dynamic feed to explore posts from other users
  • Posts categorized for easy exploration by cuisine or type
  • Advanced search and filtering options for finding specific recipes
